# rustgrid-agent v1.0.0

The first stable public artifact release of the RustGrid worker. It executes
RustGrid-assigned Codex runs in isolated Docker Sandbox microVMs, publishes
reviewable GitHub pull requests, and reports ordered progress and token
consumption to RustGrid.

## Install

On macOS with Homebrew:

```sh
brew install RustGrid/tap/rustgrid-agent
```

Checksummed native macOS and Linux archives, the packaged Rust crate, SPDX SBOM,
Homebrew formula, and attestations are attached to this release. Verify the
matching `.sha256` file and GitHub artifact attestation before installing a
native archive.

## Deployment status

Artifact publication is not production deployment approval. Before deploying
the release container, complete the repository's staging certification against
its exact image digest, including isolation, recovery, lease-loss, replay,
secret-redaction, soak, and rollback scenarios with operator and security
approval.

## Known limitations

- Windows is not supported or tested.
- Production execution requires a preflighted Docker Sandbox host and a
  digest-pinned template.
- Exact network destination policy remains an operator or organization
  governance responsibility.
- Failed workspaces may contain proprietary source and are retained according
  to the configured retention period.
- Homebrew/core inclusion is not part of this release; use `RustGrid/tap`.

See [known limitations](https://github.com/RustGrid/rustgrid-agent/blob/v1.0.0/docs/known-limitations.md)
and [production operations](https://github.com/RustGrid/rustgrid-agent/blob/v1.0.0/docs/operations.md)
for the complete operational boundary.

## Rollback

This is the first tagged release, so there is no previous public container tag.
If certification fails, stop the worker and do not deploy this image. Existing
operators should restore their previously certified internal image or
source-built commit and its matching configuration. Do not replace or mutate
the `v1.0.0` tag or release assets; publish fixes under a new semantic version.
