# Support

## Community support

Use GitHub Discussions for setup questions, deployment patterns, and design conversations. Use GitHub Issues for reproducible defects and focused feature requests.

Include the agent version, operating system, deployment model, sanitized configuration, relevant run phase, and redacted logs. Never include RustGrid API keys, GitHub tokens, repository secrets, or private source code.

## Security

Follow `SECURITY.md` for suspected vulnerabilities. Security reports posted publicly may be removed to protect users.

## Service support

This repository does not define a commercial support SLA. RustGrid customers should use the support channel in their service agreement for control-plane incidents, tenant data issues, or production outages.
